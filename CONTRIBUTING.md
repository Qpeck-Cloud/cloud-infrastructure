# Contributing

Thank you for your interest in contributing to this repository.

We welcome improvements to cloud infrastructure documentation, tutorials, scripts, deployment examples, and best practices.

## Ways to Contribute

* Improve documentation
* Fix typos or errors
* Add Linux VPS tutorials
* Add Windows VPS tutorials
* Share Docker and Kubernetes examples
* Add cloud security best practices
* Improve monitoring and automation guides

## Contribution Guidelines

* Keep content accurate and well documented.
* Use clear Markdown formatting.
* Test scripts before submitting.
* Follow security best practices.
* Keep examples simple and production-ready.

We appreciate every contribution that helps developers build better cloud infrastructure.
